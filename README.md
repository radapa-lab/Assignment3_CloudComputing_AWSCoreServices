# AWS Data Pipeline and Athena Dashboard

## Overview

This project demonstrates the implementation of an end-to-end data pipeline using AWS services. The pipeline is designed to ingest raw data, process it automatically, make it queryable, and finally visualize the results through a web-based dashboard.

The system uses Amazon S3 for storage, AWS Lambda for event-driven processing, AWS Glue for data transformation, Amazon Athena for querying, and a Flask application for displaying results.

## Architecture and Approach

The overall workflow of the system is as follows:

Raw data is uploaded into an Amazon S3 bucket.
An S3 event triggers an AWS Lambda function.
The Lambda function initiates a Glue job for processing.
The Glue job cleans and transforms the data and stores it in a processed S3 location.
A Glue Crawler scans the processed data and updates the Data Catalog.
Amazon Athena is used to query the processed data.
A Flask-based web application fetches Athena query results and displays them in a dashboard.

This architecture ensures automation, scalability, and efficient data processing.

### Step 1: Amazon S3 Bucket Structure

Two main S3 buckets (or logical folders) are used:

Raw data (landing zone): Stores incoming CSV files
Processed data: Stores cleaned and transformed data
Enriched/Athena results: Stores query outputs generated by Athena
Explanation

The S3 bucket acts as the foundation of the pipeline. Raw data is first stored, then processed data is written into structured folders, and finally Athena outputs are stored for further use.

## Conclusion

This project demonstrates the implementation of a complete data pipeline using AWS services. By integrating S3, Lambda, Glue, and Athena, the system enables automated data processing and efficient querying of large datasets.

The Flask-based dashboard provides a simple interface to visualize query results, making the processed data more accessible. Overall, this project highlights the advantages of serverless architecture, automation, and scalable data analytics using cloud technologies.
